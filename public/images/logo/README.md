# Logo images (header, top-left)

## Where to save

Put your logo files in this folder:

```
public/images/logo/
```

## File names

- **Light theme (white/light background):** `logo-2.svg` or `logo-2.png`
- **Dark theme (dark background):** `logo.svg` or `logo.png`

You can use the same file for both (e.g. one `logo.png`) if the logo works on light and dark.

## Specifications

| Spec        | Recommended        | Notes                                      |
|------------|--------------------|--------------------------------------------|
| **Format** | SVG or PNG         | SVG = sharp at any size, PNG = use 2x size |
| **Width**  | 280 px (or 140 px) | Display width is 140 px (use 2x for retina)|
| **Height** | 60 px (or 30 px)   | Display height is 30 px                    |
| **Aspect** | ~4.6 : 1           | Wide horizontal logo for the header        |
| **Background** | Transparent   | So it looks good on light and dark         |

- **PNG:** Export at **280 × 60 px** (or 140 × 30 px) for a sharp header logo.
- **SVG:** No fixed size; it will scale. Keep the design within a similar wide rectangle.

After saving, the header will use your new logo automatically.

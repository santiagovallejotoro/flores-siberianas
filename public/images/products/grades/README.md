# Main Hydrangea Grades – Product Images

Add images per grade here for the **Our Products / Main Hydrangea Grades** carousel.

## Folder structure

Each grade has its own folder; cards show pictures from that folder only.

| Folder    | Grade   | Files (examples)                    |
|-----------|---------|-------------------------------------|
| `mini/`   | Mini    | `mini-grade.jpg`, `mini-grade-1.jpg`, … |
| `select/` | Select  | `select-grade.jpg`, `select-grade-1.jpg`, … |
| `premium/`| Premium | `premium-grade.jpg`, `premium-grade-1.jpg`, … |

Update `src/components/Products/gradeImages.ts` when adding or removing files (use paths like `mini/mini-grade.jpg`).

## Image specifications

- **Dimensions:** 800×800 px or 1200×1200 px (square)
- **Format:** WebP or JPEG
- **File size:** &lt; 200 KB each
- **Subject:** Single hydrangea head showing the grade size clearly
